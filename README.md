## Usage

```sh
$ cd ~
$ git clone --recursive https://github.com/posquit0/tmux-conf ~/.tmux
$ ln -s .tmux/tmux.conf .tmux.conf
$ ~/.tmux/plugins/tpm/scripts/install_plugins.sh
```
